249 /*
250 Packet Overview
251 
252 Byte
253 0 Packet start byte
254 1 Packet indo: ID (mask 0x1F) | reserved bits (mask 0xE)
255 2 Packet counter (for ACK and retry)
256 3 Packet flags
257 
258 // packet body, may or may not exist depending on packet id - packet body is made up of 4 byte or 8 byte values.
259 4-7 Data identifier
260 8-11 Data length
261 12-15 Data offset
262 16-19 Data start
263 (n-8)-(n-5) Last piece of data
264 // end data
265 
266 n-4 Reserved
267 n-3 Checksum high byte
268 n-2 Checksum low byte
269 n-1 Packet end byte
270 */
